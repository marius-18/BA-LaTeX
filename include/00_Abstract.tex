\begin{abstract}
Ziel dieser Bachelorarbeit ist es, einen \red{effizienten} Algorithmus
zur Randomisierung massiver bipartiter Graphen zu entwickeln. Dazu 
wurde das Konzept des \gc{} für bipartite Graphen
angepasst. Es werden verschiedene Möglichkeiten besprochen, dies umzusetzen.
An Hand von Benchmarks wird die schnellste er getesteten Methoden ausgewählt.
Im Vergleich zu dem schon existierenden \gc{} Algorithmus 
werden mit dem in dieser Arbeit entwickelten Algorithmus 
auf unterschiedlichen Test Instanzen Speedups von bis zu xyz erreicht.



\end{abstract}
